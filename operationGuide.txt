補充一、
登入Firebase專案後，在左欄"建構"底下點擊"Realtime Database"即可看到資料庫全貌。
另，執行database.py前，為連接上Firebase，需先下載Firebase Admin SDK
有兩種方式：
<法一> 終端指令
在Visual Studio Code 本專案的終端，輸入指令：pip install -r requirements.txt
<法二> 手動下載
1. 確認python版本，於終端輸入：python --version
2. 為確保下載速度，避免卡頓，可先更新pip，於終端輸入：python -m pip install --upgrade pip setuptools
3. 如python為version 3，於終端輸入：python -m pip install firebase-admin
4. 若成功執行，應出現"資料寫入成功！"

補充二、
鑒於serviceAccountKey.json是重要的金鑰，各方建議不要將它存在GitHub，
故而有了以下的步驟，目的是為了在每次開機的時候自動加載路徑，
這樣在開啟並執行本專案的時候，才能自動連上資料庫：
1. 下載傳到各位信箱的serviceAccountKey.json檔案，並將它存在電腦上的風水寶地。
2. 開啟控制台(Windows+R, 輸入control)，找到系統進階設定中的環境變數。
註、建議直接在搜尋欄輸入"系統進階設定"比較快。
3. 點擊環境變數，在"系統變數"框下按新增，並輸入以下值：
(1)變數名稱：FIREBASE_CREDENTIALS
(2)變數路徑：風水寶地的路徑，建議直接利用檔案位置的搜尋功能
4. 靜候佳音，或重啟電腦，等待環境變數存進電腦，便可執行database.py
